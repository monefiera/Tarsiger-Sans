# Tarsiger-Sans
Product SansっぽいOutfit FontsとM PLUS 1 Fontsの合成フォントです。
MejiroやKoruriの系列に連なるフォントのため好きな青い鳥「ルリビタキ」から英名を取ってTarsigerとしました。
英数字はOutfit、2バイト文字はM PLUS 1になります。
